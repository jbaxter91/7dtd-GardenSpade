# GardenSpade Mod for 7 Days to Die

## Overview

GardenSpade is a mod for 7 Days to Die that enhances your gardening and farming experience. This mod introduces new a item, recipe, and progression elements to make gardening more engaging and rewarding. It is designed to be lightweight and compatible with most other mods.

**Note:** This mod uses a custom icon for the Garden Spade item, but does not include any custom 3D models. All items use existing in-game models, with the custom icon provided in the UIAtlases folder.

---

## Features
- New gardening tool and items
- Custom recipe for crafting and upgrading
- Progression integration for balanced gameplay
- Localized text for in-game clarity

---

## Installation Instructions

1. **Download the Mod**
   - Download the entire `GardenSpade` mod folder.

2. **Locate Your Mods Folder**
   - Navigate to your 7 Days to Die installation directory.
   - Open the `Mods` folder. If it does not exist, create a new folder named `Mods` (case-sensitive).
     - Example path: `C:\Program Files (x86)\Steam\steamapps\common\7 Days To Die\Mods`

3. **Install the Mod**
   - Copy the entire `GardenSpade` folder (including all files and subfolders) into the `Mods` directory.
   - The folder structure should look like this:
     ```
     Mods/
       GardenSpade/
         ModInfo.xml
         Config/
           items.xml
           recipes.xml
           progression.xml
           Localization.txt
         UIAtlases/
           ItemIconAtlas/
     ```

4. **Launch the Game**
   - Start 7 Days to Die. The mod will be loaded automatically if installed correctly.

---

## Compatibility
- This mod does **not** add any custom 3D models, ensuring maximum compatibility with other mods.
- The Garden Spade item uses a custom icon, included in the mod's UIAtlases folder.
- All new items use existing in-game models.
